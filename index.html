Privacy Policy

TVB AUTOMATION Chrome Extension
Effective Date: March 5, 2026

TVB AUTOMATION is a Chrome extension designed exclusively to automate and optimize product management workflows on The Vintage Bar website.

We are committed to your privacy.

TVB AUTOMATION collects NO personal data whatsoever — except for temporary, session-only Google authentication (if you enable personalized settings) to verify you are a real, authorized user. No data is stored, logged, transmitted to our servers, shared, or sold.

1. Data We Collect

The extension collects **zero personal or sensitive user data** in the following categories:

  • No names, email addresses, IP addresses, or other identifiers
  • No browsing history, location, or website interaction data outside The Vintage Bar site
  • No financial, health, payment, or authentication credentials (beyond temporary Google token handling)
  • No analytics, tracking, cookies, telemetry, or logs of any kind

All automation features (measurements, duplicate detection, color/material management, etc.) operate **locally in your browser** or interact **only with The Vintage Bar website** pages you actively visit. Nothing is sent externally.

2. Google Authentication (Optional – For Real User Verification)

If you enable personalized settings, the extension uses Google's secure authentication (via chrome.identity or OAuth) **solely to verify you are a legitimate user** and to fetch basic public profile information (e.g., name or picture) for tailoring features to your preferences.

We:

  • Do NOT store access tokens, refresh tokens, or any auth data
  • Process authentication data **temporarily in-memory during your active browser session only**
  • Rely entirely on Google's secure systems — we never see or handle sensitive credentials
  • Use this **only for in-extension personalization**; no data is transmitted to any server we control

This is required solely for authenticating real users and complies with Chrome's identity API guidelines. No other purpose is served.

3. Data Usage, Sharing & Security

We do not collect, use, share, sell, rent, or disclose any user data to third parties. All operations remain on your device or with The Vintage Bar's own site. Authentication data is handled securely via HTTPS through Google — we apply no additional transmission.

4. Changes to This Policy

We may update this policy occasionally. Changes will appear here with a new effective date.
